# Rogue Gear

{: .note}
> **Proficiencies**
>
> [Rogues](../../../character_creation/class/rogue) have proficiency in:
>
> * Armor: Light armor
> * Weapons: Simple weapons, hand crossbows, longswords, rapiers, shortswords
