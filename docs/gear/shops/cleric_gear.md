# Cleric Gear

{: .note}
> **Proficiencies**
>
> [Clerics](../../character_creation/class/cleric) have proficiency in:
>
> * Armor: Light armor, medium armor, [heavy armor](../../../data/archetypes/cleric_life), shields
> * Weapons: All simple weapons
> * Tools: Holy symbols, spell scrolls
>
