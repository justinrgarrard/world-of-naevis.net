# Wizard Gear

{: .note}
> **Proficiencies**
>
> [Wizards](../../character_creation/class/wizard) have proficiency in:
>
> * Armor: None
> * Weapons: Daggers, darts, slings, quarterstaffs, light crossbows
> * Tools: Wands, spellbooks, spell scrolls
